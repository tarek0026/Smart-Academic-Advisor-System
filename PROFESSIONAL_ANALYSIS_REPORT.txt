          SMART ACADEMIC ADVISOR SYSTEM - ANALYSIS REPORT

1. PROJECT OVERVIEW

The Smart Academic Advisor System is a Java-based application that provides 
intelligent course recommendations for Computer Science and AI students. It 
analyzes student academic profiles and generates personalized semester plans 
while respecting prerequisites, GPA limits, and track specialization.

Purpose: Help students select suitable courses based on academic progress, 
prerequisites, and specialization track (Big Data, Media, General, or AI).


2. AVAILABLE COURSES LOGIC

- Prerequisite Checking: Only courses with all prerequisites completed are 
  recommended.
- Completed Courses Excluded: Courses already completed are not recommended 
  again.
- Track-Based Filtering: Courses are filtered by track (BIGDATA, MEDIA, GENERAL 
  for CS; AI for AI major).
- GPA Restrictions: Special courses (internships, graduation projects) require 
  minimum GPA thresholds.


3. RECOMMENDATION SYSTEM LOGIC

The system prioritizes courses using a scoring system:

- Tier 1: Delayed courses (Score +1000) - courses from previous semesters
- Tier 2: Current semester courses (Score +500) - courses scheduled for now
- Tier 3: Future courses (Score +100) - upcoming important courses

Dependency Scoring: Courses unlocking many future courses get +50 points per 
dependent course, ensuring students take gateway courses early.

Sorting: All courses sorted by score descending. Highest priority courses 
recommended first.


4. GPA-BASED LOAD ANALYSIS

Maximum credit hours per semester based on GPA:

- Half Load: GPA < 2.0 → Max 12 credits (academic warning status)
- Normal Load: 2.0 ≤ GPA ≤ 3.0 → Max 19 credits (standard standing)
- Overload: GPA > 3.0 → Max 21 credits (excellent standing)

The system respects these limits when building recommendations, preventing 
overload for struggling students while enabling acceleration for high performers.


5. ELECTIVE RECOMMENDATION GROUPS

Rather than auto-selecting electives, the system groups them for student choice:

Big Data Track: "Choose 1 Big Data Elective" + "Choose 3 from General Pool"
Media Track: "Choose 1 Media Elective" + "Choose 3 from General Pool"
General Track: "Choose 3 from General Pool" (maximum flexibility)
AI Major: "Choose AI Electives"

Benefits: Provides flexibility and realism while maintaining academic guidance.


6. OUTPUT STRUCTURE

The system generates RecommendationBlocks:

- FIXED Type: Required courses recommended based on priority
- CHOICE Type: Elective options for student selection

Example Output:
Block 1 (FIXED): "Recommended Courses" - [CS301, CS305, CS310]
Block 2 (CHOICE): "Choose 1 Big Data Elective" - [CS401, CS402, CS403]
Block 3 (CHOICE): "Choose 3 from General Pool" - [MATH401, CS420, CS421]


7. OBJECT-ORIENTED PROGRAMMING CONCEPTS

- Classes & Objects: Student and Course classes with encapsulation
- Encapsulation: Private fields with validated setters (GPA 0.0-4.0, valid IDs)
- Abstraction: LoadService hides complexity of load calculations
- Modularity: Separate packages (Classes, Services, Application, Testing)
- Service Architecture: RecommendationService, LoadService, AdvisorServices 
  work independently and are easily testable


8. DEPENDENCY ANALYSIS

Courses are scored by their dependency count:
- Gateway courses (prerequisites for 5+ courses): High priority
- Foundation courses (prerequisites for 2-4 courses): Medium priority
- Terminal courses (prerequisites for 0-1 course): Lower priority

Example: CS201 (Data Structures) unlocks CS301, CS302, CS303, CS401, CS402 
→ Receives +250 point boost (5 × 50)


9. ADVANTAGES OF THE SYSTEM

✓ Prevents prerequisite violations
✓ Automates routine advising 24/7
✓ Respects workload capacity (GPA-based)
✓ Prioritizes course dependencies
✓ Provides flexible elective grouping
✓ Maintains academic standards

10. CONCLUSION

The Smart Academic Advisor System successfully combines intelligent algorithms 
with clean object-oriented design. It provides automated, personalized academic 
guidance that respects student capacity and curriculum requirements.

Key achievements:

- Intelligent prioritization using multi-factor scoring

The system demonstrates how modern software engineering principles enable
