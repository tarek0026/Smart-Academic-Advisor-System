📘 Smart Academic Advisor System – Development Plan
🧠 Phase 1: Problem Understanding & Requirements

The system aims to help students plan their academic schedule efficiently.

Functional Requirements:
Accept student input:
Name
ID
GPA
Completed courses
Determine the appropriate academic load:
Half Load / Normal Load / Overload
Display:
Available Courses
Recommended Courses (based on roadmap)
Handle invalid inputs and errors properly


🗂️ Phase 2: Data Design
Course Structure:

Each course should include:

Course Name
Course Code
Prerequisites
Semester (based on roadmap)
Credit Hours

Roadmap:
Define courses for each semester
This will be used to determine recommended courses


Student Structure:
ID
Name
GPA
Completed Courses



⚙️ Phase 3: System Rules (Logic)
Rule 1: Load Determination
Low GPA → Half Load
متوسط GPA → Normal Load
High GPA → Overload

(Define exact GPA ranges and number of courses for each case)


Rule 2: Available Courses

A course is considered available if:

All its prerequisites are completed
The student has not taken it before




Rule 3: Recommended Courses (Enhanced)

Recommended courses are selected based on availability, roadmap alignment, and priority rules.

Step 1: Base Filtering

A course is considered for recommendation if:

It belongs to the current semester in the roadmap
AND it is available (all prerequisites are satisfied)
Step 2: Handling Delayed Courses (Higher Priority)
Any course from previous semesters that the student has not completed
AND is available

➡️ These courses should be prioritized over current semester courses

Step 3: Priority Optimization (Smart Selection)

If there is a conflict or limited load, courses are selected based on:

Delayed courses (from previous semesters) → Highest priority
Courses from the current semester
Other available courses



🔄 Phase 4: System Flow
Receive student input
Validate inputs
Determine load type
Retrieve all courses
Filter available courses
Filter recommended courses
Adjust number of courses based on load
Display results
